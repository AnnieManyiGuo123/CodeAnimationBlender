# CodeAnimationBlender
# Exploration Project #1 — Coding for Animation

A project exploring 3D animation in Blender using code and potential AI applications, aiming to create animations and models through programming rather than manual tools.

---

## Features

- Create, move, and rotate objects
- Build scenes with camera and lights
- Assign textures, colors, and materials to objects
- Render animations
- Coding language applications (lists, for-loops, etc.)
- Blender MCP integration

---

| Technology | Version |

## Installation

### Prerequisites

- Python `3.13` or higher
- Blender `3.0`+ or Blender `2.83 LTS`
- Claude `1.1.4498 (24f768)` or higher
- UV package manager

### Steps — Running the Animation

1. Open the repository: [CodeAnimationBlender](https://github.com/AnnieManyiGuo123/CodeAnimationBlender.git)
2. Copy the script code
3. Open Blender and navigate to the **Scripting** tab
4. Paste the code into the editor
5. Click **Run Script**

### Steps — Blender MCP Setup

Follow the README instructions here: [blender-mcp](https://github.com/ahujasid/blender-mcp.git)

---

## Usage

1. Navigate the script in Blender's **Scripting** tab
2. Press the **Run Script** button
3. To render and export the animation:
   - Click **Output** in the Properties panel to set render settings
   - Choose an output location
   - Click the **Render** tab in the top-left menu
   - Select **Render Animation**
   - Check your chosen output location for the result
